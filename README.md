# 🖐️ Hand Gesture Volume Control

## 📌 Project Overview
This project uses **Computer Vision + AI** to control system volume using hand gestures captured from webcam.

---

## 🚀 Features
- Real-time hand tracking using webcam
- Finger distance-based volume control
- Smooth gesture recognition
- Works on Windows system volume

---

## 🛠️ Tech Stack
- Python
- OpenCV
- MediaPipe
- PyAutoGUI

## 🧠 Project Explanation
This project detects hand gestures in real-time using webcam and controls system volume based on finger distance using AI computer vision.


## ⚙️ How it Works
1. Webcam captures hand movement  
2. MediaPipe detects hand landmarks  
3. Distance between thumb and index finger is calculated  
4. Based on distance:
   - Increase → Volume Up 🔊  
   - Decrease → Volume Down 🔉
  


  ## 📊 Project Highlights
- Real-time AI processing  
- No external hardware required  
- Smooth gesture detection  
- Beginner-friendly computer vision project

## 🧩 System Flow
Camera → Hand Detection → Landmark Tracking → Distance Calculation → Volume Control
